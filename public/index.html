Darklang

Darklang is a beautiful functional OSS language, with easy-to-use modern tooling, so you can enjoy writing code again

Influenced by: Rust, Elm, Typescript, Rescript, OCaml, F#, Python, Ruby, Clojure

At a glance

  • A modern functional language
    • Simple static types based around simple immutable types, composition via records and enums, Option and Result types, garbage collection, async runtime, and Unicode strings/chars.
    • Gradual static typing, and other language and parser features, to allow fast prototyping (code does not need to compile or type check to run, and starts instantly in our interpreter)
    • No compile step, no null, no exceptions, no inconsistent function names, no weird equality conditions, no async/await keywords, no inheritence
  • Modern Packaging system
    • Always on, syncs code to package namespaces automatically
    • Replaces Git, Github, Docker, and removes installation step, environments
    • Individually versioned functions, constants and types (use multiple versions of the same function or type in the same program), allows for safe and easy migrations
    • Easily pre-release code
  • Run from anywhere
    • CLI runs any packages with no install: darklang @username.functionName arg1 arg2 from the command line (or the browser, slack, discord, vs code, etc)
    • OptionalInstant cloud deployment of code, with instant creation of DBs, http endpoints, and worker queues, with no containers, no cold starts, no orchestration, or other devops/cloud engineering required.
    • Optional Or deploy to your own cloud
  • Designed for AI
    • We’ve adapted our tools and approach to developing dark to recognize the significance of GenAI in the development process
    • Designed to enable generated programs. Run partial or incorrect programs
    • Exposed language internals, parser, error messages.
    • Build short CLI programs from prompts darklang prompt "find all js files which don't have a CSS file of the same name"
    • Build vendor SDKs from prompts and OpenAPI docs
    • Build complex programs with AI agents
    • Works well with GitHub Copilot, VS Code, etc.
  • Open source
    • In 2023, we expect to fully open source Darklang, which is currently source available
